# 📦 Installation

Clone the repository:

```bash
git clone https://github.com/NikulPrajapati55/tailwind-css.git
```

Install dependencies:

```bash
composer install
npm install
npm run dev
```

Run Laravel server:

```bash
php artisan serve
```
